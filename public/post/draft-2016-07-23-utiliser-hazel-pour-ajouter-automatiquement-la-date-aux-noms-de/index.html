Ajouter automatiquement la date aux noms de fichiers avec Hazel

Hazel fait partie de mes outils d’automatisation indispensables avec Keyboard Maestro, Quicksilver et Applescript. Je vais montrer ici comment ajouter automatiquement la date du jour à un nom de fichier avec Hazel.

Par exemple, je nomme les articles de ce blog dans Hugo avec la nomenclature suivante :

2016-06-02-article-un.md
2016-07-10-toto.md
etc.

Il est cependant plus facile et efficace dans Hugo de taper hugo new /post/toto.md plutôt que hugo new /post/2016-07-10-toto.md. J’utilise docn la méthode suivante : * Je crée l’article avec hugo new /post/toto.md * Hazel détecte la création d’un nouveau fichier article, ajoute la date du jour au début 2016-07-10-toto.md, puis ouvre automatiquement le fichier dans mon éditeur de texte préféré, IA Writer.

Pour ce faire, j’utilise la règle suivante dans Hazel : ![Règle Hazel pour ajouter la date à un nom de fichier][img]

Ce n’est qu’un petit exemple des possibilités de Hazel. Parmi mes autres utilisations j’ai : * Une règle pour tagger automatiquement les dossiers du Finder qui correspondent à un projet actif dans Omnifocus. * Des règles pour numéroter et tagger automatiquement les documents scannés en fonction de leur contenu (RSI, banque, etc.) - le classement de mon courrier est pratiquement instantané ! * Des règles pour trier automatiquement les téléchargements en fonction de leur type.
